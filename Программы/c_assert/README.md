# 21school_piscine-c_assert #

Если хотите прогнать вашу функцию на тесты, то:
1. Поместите в папку с `test(day)_(ex).c` вашу функцию `ft_(yourfunc).c`
2. Скомпилируйте и запустите `gcc -Wall -Wextra -Werror test* -o test && ./test"`
   * Альяс через `testd` команду `alias testd="gcc -Wall -Wextra -Werror test* -o test && ./test"`

Функция assert очень проста в использовании, вы всегда сможете добавить сами свой вариант проверки и ожидаемый ответ.
Если этот репозиторий вам помог, то не забудьте предложить свои варианты тестов, которые помогут в свою очередь другим.
